# **App Name**: RoleHub Lite

## Core Features:

- Splash Screen: Splash screen to briefly display the logo upon app launch.
- Login Screen: Login screen with 'E-mail' and 'Password' input fields.
- Event Feed: Event feed displaying cards with event details. Incorporates a search input field for event discovery and filter icons.
- Event Detail: Detailed view of an event, displaying full-width images, title, date/time, location, and a brief description.
- Event Save: Provide an action to favorite, or save event for future reference.
- User Profile: Profile screen displaying user information with options to edit name and avatar.
- Suggested events: Utilize an AI tool that suggests events which are personalized for the user, by comparing details from previously chosen events. 

## Style Guidelines:

- Primary color: Neon Green (#00FFA3) for primary buttons and highlights, evoking energy and a modern feel.
- Background color: Dark background (#111111) for a modern and immersive feel.
- Accent color: Electric Blue (#0055FF) for secondary accents, providing a cool, technological contrast.
- Titles font: 'Poppins' (sans-serif) in bold for titles, providing a modern and readable look. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) in regular for body text, providing a clear and contemporary feel. Note: currently only Google Fonts are supported.
- Minimalist and outlined icons, with neon green as the highlight color.
- Mobile-first layout (375x812 px), ensuring touch targets are at least 44x44 px.
- Subtle animations for transitions between screens with a slide left/right of 250 ms with an ease effect.
